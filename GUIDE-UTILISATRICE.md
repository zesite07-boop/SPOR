# Guide Utilisatrice · Serey Padma

Ce guide t'aide a installer Serey Padma sur telephone/tablette et a utiliser rapidement les modules essentiels.

## 1) Installation sur telephone / tablette

### iPhone / iPad (Safari)

1. Ouvre l'URL de l'application dans Safari.
2. Appuie sur `Partager`.
3. Choisis `Sur l'ecran d'accueil`.
4. Valide le nom `Serey Padma`, puis `Ajouter`.
5. Ouvre l'app depuis l'icone pour profiter du mode plein ecran.

### Android (Chrome)

1. Ouvre l'URL de l'application dans Chrome.
2. Appuie sur `Installer l'application` ou `Ajouter a l'ecran d'accueil`.
3. Confirme l'installation.
4. Lance l'app depuis l'icone.

## 2) Bien demarrer

- Si Supabase est configure, connecte-toi via magic link (`/connexion`).
- Si Supabase n'est pas configure, tu peux quand meme utiliser l'app en mode local (Dexie + PWA).
- Le badge `Mode demo · local` peut etre affiche ou masque selon la configuration.

## 3) Modules principaux

## Accueil

- Vue synthese du sanctuaire.
- Raccourcis vers les 6 onglets.
- Calendrier cosmique, oracle du jour, energie quotidienne.

## Ops / Logistique

- Gestion retraites, chambres, listes et suivi operationnel.
- Organisation des taches avec interface mobile fluide.

## Oracle / Bien-etre

- Tirages, interpretation et guidance.
- Vision claire des energies du moment.

## Rayonner (Marketing)

- Queue editoriale.
- Generation de textes pour Instagram/TikTok.
- Copie rapide et export image.

## Jeux / Oracle ludique

- Espace plus leger pour relancer l'elan creatif.
- Compatible Mode Hyperfocus.

## Tresor

- KPIs, simulations et pilotage economique.
- Aide a la decision avec une presentation douce.

## 4) Mode Hyperfocus

- Active le bouton `Hyperfocus` dans les ecrans qui l'affichent.
- Raccourcis clavier:
  - `Ctrl+Shift+H` : activer/desactiver Hyperfocus
  - `Esc` : quitter Hyperfocus
  - `Alt+1..6` : naviguer entre les 6 onglets

## 5) Generer du contenu marketing

1. Va dans `Rayonner`.
2. Choisis un `Format` et un `Scenario editorial`.
3. Clique sur `Generer`.
4. Verifie la previsualisation.
5. Clique `Copier` pour coller dans Instagram/TikTok, ou `Telecharger image`.

Astuce: en mode offline, la generation locale reste disponible.

## 6) Conseils test mobile reel

- Teste en mode avion pour verifier la continuite offline.
- Verifie les transitions entre onglets et la facilite de touch.
- Fais au moins un test iOS et un test Android avant go-live.
